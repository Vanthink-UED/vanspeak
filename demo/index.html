How To Use It?

快速开始

<script src="./build/vanspeak.min.js"></script>

使用CDN地址

<script src="http://s1.vued.vanthink.cn/vanspeak.min.js"></script>

In your JS File

vanspeak.say('Hello wolrd');

Examples

拼读一个单词

vanspeak.say('Hello wolrd');

拼读一个句子

vanspeak.say('Visit BBC News for up-to-the-minute news');

停止当前阅读

vanspeak.cancel();


拼读多个单词

vanspeak.say('Hello wolrd');
